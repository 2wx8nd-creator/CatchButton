# C# 강의 2주차 과제 : 버튼을 잡아라!!

-버튼에 커서가 enter 시, 버튼이 랜덤하게 이동.  
-클릭 성공 시, 화면 변경(성공), 다음 레벨(버튼의 이동 시간 감소) 도전 가능  
-클릭 실패 시, 화면 변경(실패)
